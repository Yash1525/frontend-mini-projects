# frontend-mini-projects

This repository contains a collection of beginner-friendly HTML & CSS mini projects designed to demonstrate core front-end development concepts, UI design techniques, and responsive layouts.

##  Features

- Styled buttons with hover effects and gradients  
- Basic webpage structure with external CSS  
- Responsive food menu using CSS Grid & Flexbox  
- Parallax scrolling effect for modern UI design  
- Tribute page with clean layout and responsiveness  
- Mobile-friendly designs using media queries  

##  Project Structure

- index.html        # Basic webpage
- styles.css        # External styling
- buttons.html      # Button designs
- menu.html         # Food menu layout
- parallax.html     # Parallax design
- tribute.html      # Tribute page


##  Learning Outcomes

- Understanding of HTML structure and semantics  
- Hands-on experience with CSS styling and layouts  
- Implementation of responsive design techniques  
- Creating interactive UI elements with hover effects  
- Building visually appealing web pages

##  Purpose

This project is created to practice and demonstrate fundamental front-end development skills using simple and practical examples.
